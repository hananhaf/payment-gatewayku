QRIS Payment

Notification Listener

Aplikasi Android yang meneruskan notifikasi "dana masuk" dari aplikasi bank/QRIS di HP kasir ke payment gateway — supaya pembayaran tercocokkan otomatis.

Unduh APK

Android 8.0+ · ~9.3 MB · tertandatangani

Langkah Pemasangan

  1. Unduh lalu buka file APK. Jika muncul peringatan, izinkan "Instal aplikasi tak dikenal" untuk browser Anda, lalu lanjutkan.
  2. Buka aplikasi Notification Listener.
  3. Beri izin Akses Notifikasi saat diminta (Setelan → Akses notifikasi → aktifkan aplikasi ini).
  4. Isi Endpoint URL dengan alamat webhook merchant Anda:
    https://payment.bppunh.com/webhook/<id-merchant>
    lalu API Key merchant Anda. Simpan.
  5. Pastikan HP tetap online dan matikan optimasi baterai untuk aplikasi ini agar terus berjalan di latar belakang.
API Key setara password merchant — jangan dibagikan atau dipasang di HP milik orang lain.

Butuh: Android 8.0+, koneksi internet aktif, dan satu aplikasi bank/e-wallet yang mengirim notifikasi transaksi.
